ProEsporte Sabará

O ProEsporte Sabará é uma plataforma web criada para estruturar as atividades disponibilizadas pela prefeitura, tornando-as
mais acessíveis e transparentes para cada cidadão.

Nosso objetivo geral é capacitar o cidadão a adotar uma rotina de treino segura, otimizando o uso dos espaços públicos.
Para cumprir o prazo de um único semestre, nosso Produto Mínimo Viável (MVP) foca em duas funcionalidades essenciais na 
forma de uma Aplicação Web responsiva
