## Olá Dev!!! Neste tópico, vou mostrar como você deve realizar a configuração do sistema na sua máquina.. 

###  🔹 1. Primeiro passo — antes de iniciar o projeto, é essencial configurar corretamente o servidor Apache. Sem essa etapa, o sistema não conseguirá responder às requisições HTTP nem servir os arquivos da aplicação.
➡️ Acesse o tutorial aqui:
- [XAMPP](./ConfiguraçãoXAMPP.md)
###  🔹 2. Segundo passo — configurar o banco de dados. Antes de qualquer coisa, você precisa seguir o tutorial de configuração do banco de dados. Sem essa etapa, o sistema não conseguirá se conectar ao MySQL.:
➡️ Acesse o tutorial aqui:
- [Banco de Dados](./ConfiguraçãoBancoDeDados.md)
### 🔹 3. Terceiro passo — executar os comandos no terminal : Após finalizar toda a configuração do banco, você deverá rodar alguns comandos no terminal do VS Code para iniciar o sistema.
➡️ Guia completo dos comandos:
- [Comando no terminal](./RodandoSistema.md)
